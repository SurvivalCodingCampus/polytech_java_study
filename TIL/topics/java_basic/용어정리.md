## Git 용어 정리
 - git init - 깃 초기화
 - git clone - 복제
 - git status - 작업 디렉토리와 스테이징 내역 확인
 - git add - 파일 추가
 - git reset - 커밋내역 삭제
 - git commit -m "커밋할 내용" - 로컬 저장소에 등록
 - git log - 기록확인
 - git rm - 삭제 (add시 잘못한 부분 삭제)
 - rm -rf (파일 명) - 파일 삭제
 - git push - commit한 내용을 원격저장소에 올리기
 - git pull - 원격 저장소에서 로컬 저장소로
