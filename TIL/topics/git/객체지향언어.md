###객체지향언어
- 용어
    - 오브젝트 (Object ) : 모든 객체
    - 클래스 ( Class ) : 오브젝트를 추상화하여 필드와 멤버로 구현한 틀
    - 인스턴스 ( Instance ) : 인스턴스화로 생성 된 객체
    - 인스턴스화: 참조 변수와 new 키워드를 사용하여 오브젝트(객체)를 생성하는 방법
- 장점 : 절차지향보다 유지 보수가 쉽고 안전성이 높아 버그를 줄일 수 있기 때문
- 동적 메모리 할당 : 프로그램 실행( 런타임 ) 중에 메모리를 할당하는 것

'''
System.out.println();
